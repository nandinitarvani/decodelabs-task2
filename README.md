# 💰 Project 2 – Expense Tracker

This project was developed as part of my internship at DecodeLabs.

# 📌 Project Overview

A Python-based Expense Tracker that allows users to record expenses, calculate total spending, and analyze their expenses through a simple console application.

# 🚀 Features

* Add multiple expenses
* Calculate total expenses
* Count the number of expenses entered
* Calculate average expense
* Display highest and lowest expenses
* Input validation using exception handling
* Store expenses using Python lists

# 🛠 Technologies Used

* Python
* Lists
* Loops
* Conditional Statements
* Exception Handling
* Mathematical Operations

# 📚 Learning Outcomes

This project strengthened my understanding of accumulators, data processing, list operations, exception handling, and real-world problem-solving using Python.
